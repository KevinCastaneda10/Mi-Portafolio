Mis Tecnologías Favoritas

Estructura de páginas web

  • HTML

Si bien esta tecnología es importante para estructurar una página web, es una de las primordiales al momento de crear y desarrollar una aplicación, ya que es desde donde se crea "el esqueleto" de la página.

Estilos de páginas web

  • CSS

Esta tecnología es la que nos ayuda a dar cuerpo a nuestra página a través de estilos con base a un requerimiento o un proyecto asignado. Al igual que HTML, es importante al momento de desarrollar una aplicación web.

Interfaz de usuarios

  • JavaScript

Este lenguaje de programación, al igual que las tecnologías anteriores, es de suma importancia, ya que acompañado de varias tecnologías más, son los que nos ayudan a que nuestra página tenga una mejor interfaz de cara al usuario.
